# 🎧 Spotify Clone (HTML & CSS)

A modern and responsive front-end clone of the Spotify web player built using pure HTML and CSS. This project replicates the layout, sidebar navigation, music player UI, and dark theme design of Spotify.

---

## 🚀 Features

* 🎵 Sidebar navigation (Home, Search, Library)
* 🧭 Clean and modern UI layout
* 🎧 Music player interface (play bar)
* 📱 Responsive design
* 🌙 Dark theme inspired by Spotify
* 🎯 Card-based music sections (Recently Played, Trending)

---

## 🛠️ Technologies Used

* HTML5
* CSS3
